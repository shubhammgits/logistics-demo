# Templates

Reusable page skeletons to speed up new client builds:
- `landing/` — marketing landing page skeleton
- `portfolio/` — scroll-storytelling portfolio (uses the 3D skill by default)
- `saas-marketing/` — SaaS product marketing site skeleton
- `agency/` — agency/studio site skeleton

Each should be a working Next.js route group the agent can copy and re-skin
using `design-system/DESIGN.md`, not a from-scratch generation every time.
